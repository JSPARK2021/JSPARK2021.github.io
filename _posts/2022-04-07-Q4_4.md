# JAVA Quiz 4-4

**JAVA Quiz 4-4**, 자바 퀴즈 4-3 입니다..

```java
package java_study;

public class Q4_4 {

	public static void main(String[] args) {
		//for문을 사용해 1부터 100까지의 숫자를 출력해 보자.
		
		for(int i=0; i<101; i++) {
			System.out.println(i);
		}

	}

}
```
