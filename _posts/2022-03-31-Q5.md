# JAVA QUIZ#5

**JAVA QUIZ#5**, 자바 퀴즈5번 입니다..

```java
package java_study;

public class Q5 {

	public static void main(String[] args) {
		//다음과 같은 문자열 a:b:c:d가 있다. 
		//문자열의 replace 함수를 사용하여 a#b#c#d로 바꿔서 출력해 보자.
		
		//주어진 문자열
		String a = "a:b:c:d";
		System.out.println(a.replaceAll(":", "#")); // : 를 #로 바꿔서 출력함
		
		
     
	}

}
```
